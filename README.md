# Python Programming

A learner-friendly Python curriculum organized from **foundations** to **advanced topics**.

## Prerequisites
- Python 3.x installed
- A code editor (VS Code, PyCharm, or similar)
- Basic command-line familiarity

## Learning Structure
Follow modules in order:

### 00-foundations
1. `01-variables-and-data-types`
2. `02-operators-and-expressions`
3. `03-control-structures`
4. `04-functions-basics`

### 01-intermediate
5. `05-file-io`
6. `06-exceptions-handling`
7. `07-decorators`
8. `08-built-in-functions`
9. `09-isinstance-type-checking`

### 02-advanced
10. `10-object-oriented-programming`
11. `11-numerical-programming`
12. `12-os-and-system-operations`

## Projects
- `projects/` contains practical exercises and mini projects.

## Resources
- `resources/examples/` contains sample support files (e.g., JSON/text files).
- `resources/utils/` is reserved for reusable helper utilities.

## Suggested Study Flow
Start with the foundations, complete each module in sequence, and then move to projects for hands-on practice.

For a detailed curriculum and progression, see [LEARNING_PATH.md](LEARNING_PATH.md).

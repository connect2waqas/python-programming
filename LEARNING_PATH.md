# Learning Path

This guide provides the recommended order for learning Python in this repository.

## Phase 1: Foundations
- [ ] `00-foundations/01-variables-and-data-types`
- [ ] `00-foundations/02-operators-and-expressions`
- [ ] `00-foundations/03-control-structures`
- [ ] `00-foundations/04-functions-basics`

**Outcome:** Build confidence with syntax, core language concepts, and basic program flow.

## Phase 2: Intermediate
- [ ] `01-intermediate/05-file-io`
- [ ] `01-intermediate/06-exceptions-handling`
- [ ] `01-intermediate/07-decorators`
- [ ] `01-intermediate/08-built-in-functions`
- [ ] `01-intermediate/09-isinstance-type-checking`

**Outcome:** Learn practical coding patterns used in real scripts and applications.

## Phase 3: Advanced
- [ ] `02-advanced/10-object-oriented-programming`
- [ ] `02-advanced/11-numerical-programming`
- [ ] `02-advanced/12-os-and-system-operations`

**Outcome:** Work with advanced abstractions, computational techniques, and system-level utilities.

## Practice Track
- [ ] Complete at least one project from `projects/`
- [ ] Revisit previous modules to refactor and improve old exercises

## Recommended Progression Rules
1. Finish each module before moving to the next.
2. Run and modify every example you read.
3. Keep notes of errors and fixes while practicing.
4. Build mini-projects after each phase to reinforce learning.
